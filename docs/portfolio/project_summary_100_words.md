A laptop-runnable analytics platform demonstrating end-to-end product analytics and AI Agent observability. The project generates synthetic data (7 tables, 1M+ records) with 10 embedded anomalies, builds a 4-layer DuckDB warehouse via 41 dbt models (7 staging, 13 intermediate, and 21 mart models), serves 20 SQL business queries, provides an 8-page Streamlit dashboard with 30 charts, and implements a full A/B experiment pipeline with 5000 bootstrap iterations. All outputs are deterministically reproducible and machine-verified through structured JSON audit evidence. Built entirely with Python, DuckDB, dbt, and Streamlit. All data is synthetic.
