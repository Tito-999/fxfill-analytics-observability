# Pre-portfolio-v1.2.12 Development Acceptance Records

These files are development acceptance reports produced before the
`portfolio-v1.2.12` release.  Their test counts, commit SHAs, failure
states, and object counts describe intermediate development states and
do not represent the current repository.

- **Current release evidence:** `reports/portfolio/releases/portfolio-v1.2.12/`
- **Current acceptance reports** are regenerated by:
  - `scripts/verify_core_release.py`
  - `scripts/verify_portfolio_release.py`
